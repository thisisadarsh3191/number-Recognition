# Number Recognition 🔢



Welcome to the **Number Recognition** project! This repository provides a straightforward machine learning setup designed to recognize and classify numbers using Convolutional Neural Networks (CNNs). Whether you want to quickly test out a pre-trained model or dig into the neural network architecture, you will find what you need right here.

## 📁 What's Inside?

Here is a quick breakdown of the core files included in this repository:

* **`cnn.py`**: The blueprint of the project. This file contains the code to build and structure the Convolutional Neural Network.


* **`classification.py`**: The previous operational script. You can use this file to see a primitive classification model.


* **`cnnIdentify.keras`**: A saved, pre-trained Keras model. You can load this directly into your code to identify numbers without needing to train the network from scratch. This is the more powerful, accurate model


* **`identifyNumbers.keras`**: An additional saved Keras model containing trained weights for number recognition tasks. This is the first model, from classification.py



## 🚀 Getting Started

To get up and running with this project, follow these general steps:

1. **Prerequisites:** Ensure you have Python installed, along with the necessary machine learning libraries refer `requirements.txt`.
2. **Run a Prediction:** Just run whiteboard.py and try it out


3. **Train from Scratch (Optional):** If you prefer to train the model yourself, experiment with different layers, or learn how the architecture is built, dive into the `cnn.py` file.
